# CompraCerta — protótipo funcional

Protótipo navegável em HTML, CSS e JavaScript puros. Abra `index.html` em um navegador para começar.

## Estrutura

- `index.html`: login
- `cadastro.html`: cadastro de usuário
- `listas.html`: minhas listas
- `lista-form.html`: criação/edição de lista
- `lista-detalhes.html`: detalhes e marcação de itens
- `item-form.html`: inclusão/edição de item
- `categorias.html`: CRUD de categorias
- `produtos.html`: CRUD de produtos
- `compartilhamento.html`: participantes e convites
- `css/styles.css`: tema responsivo compartilhado
- `js/app.js`: interações e dados simulados em `localStorage`
- `images/`: recursos visuais referenciados pelas páginas

O protótipo não usa backend. Alterações em categorias, produtos e participantes ficam salvas somente no navegador.
