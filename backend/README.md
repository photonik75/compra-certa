# Backend

Diretório reservado para a API Spring Boot do CompraCerta.

Stack definida:

- Java 21;
- Spring Boot;
- Spring Web;
- Spring Security;
- Spring Data JPA com Hibernate;
- PostgreSQL.

O contrato inicial está em `../docs/api/openapi.yaml` e os testes planejados para a EF-01 estão em
`../docs/testes/testes-ef01-backend.md`.
